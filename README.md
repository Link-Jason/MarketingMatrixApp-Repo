### App Function
The application allows users to input product or campaign data to generate visual output in the Product Performance Matrix:

| Input Category    | Example Inputs                        |
|------------------|--------------------------------------|
| Product Metrics   | Margin, Market Share, Growth Rate     |
| Campaign Metrics  | Marketing Spend, Engagement, Conversion Rate |

**Output:**  
Products are mapped into quadrants (Stars, Cash Cows, Question Marks, Dogs), providing a visual guide for marketing prioritization, investment decisions, and strategic action.

### Tech Stack & Structure

- **Frontend:** React, Next.js, Tailwind CSS  
- **Components:** `MatrixChart.jsx`, `OutputPanel.jsx`, `MenuItemList.jsx`  
- **Logic:** JavaScript (core dashboard logic in `MatrixChart.jsx` and `OutputPanel.jsx`)  
- **Deployment:** Vercel  
- **Tools:** Cursor, AI-assisted prototyping, no-code integration  
- **Configuration & Utilities:** ESLint, TypeScript, PostCSS
